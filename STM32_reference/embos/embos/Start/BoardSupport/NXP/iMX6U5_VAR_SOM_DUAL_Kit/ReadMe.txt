ReadMe.txt for the NXP (Freescale) iMX6U5 start project.

This project was built for emIDE V2.20.

Supported hardware:
===================
The sample project for the iMX6U5 is prepared to run on
a Variscite iMX6U5 board. Using different target
hardware may require modifications.

Configurations:
===============
- Debug:
  This configuration is prepared for download into internal
  RAM using J-Link. An embOS debug and profiling library
  is used.

- Release:
  This configuration is prepared for download into internal
  RAM using J-Link. An embOS release library is used.

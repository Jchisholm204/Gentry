Readme.txt for the Atmel AT91SAM9261 start project.

This project was built for emIDE V2.20.

Supported hardware:
===================
The sample project for AT91SAM9261 is prepared to run on
an Atmel AT91SAM9261-EK eval board. Using different target
hardware may require modifications.

Configurations:
===============
- Debug_iRAM:
  This configuration is prepared for download into internal
  RAM using J-Link. An embOS debug and profiling library is
  used.

- Release_iRAM:
  This configuration is prepared for download into internal
  RAM using J-Link. An embOS release library is used.

ReadMe.txt for the ST STM32U545 start project

This project was built for STM32CubeIDE V1.12.1.

Supported hardware:
===================
The sample project for the STM32U545 is prepared to run on an ST NUCLEO-STM32U545RE-Q board.
Using different target hardware may require modifications.

Configurations:
===============
- Debug:
  This configuration is prepared for download into internal Flash using ST-Link.
  An embOS debug and profiling library is used.

- Release:
  This configuration is prepared for download into internal Flash using ST-Link.
  An embOS release library is used.
